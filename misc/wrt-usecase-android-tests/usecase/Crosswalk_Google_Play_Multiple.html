Test steps:

  1. Package an web app for arm platform with command: python make_apk.py --name=test --package=org.xwalk.test --app-url=http://www.baidu.com --app-versionCodeBase=1 --app-version=1.0.0 --arch=arm
  2. Package an web app for x86 platform with command: python make_apk.py --name=test --package=org.xwalk.test --app-url=http://www.baidu.com --app-versionCodeBase=1 --app-version=1.0.0 --arch=x86
  3. Login the Google Play Developer Console in Browser and click the button "Add new application"
  4. Input a Title as application name and click the button "Upload APK"
  5. Choose ALPHA TESTING or BETA TESTING and Click the button "Upload new APK to Beta", then upload the "arm" arch apk into the application
  6. Click the button "SWITCH TO ADVANCED MODE"
  7. Click the button "Upload new APK to Beta", then upload the "x86" arch apk into the same application
  8. Click the link "Manage list of testers", then add Google Groups or Google+ Communities
  9. Enter the required information in "Store Listing" "Pricing&Distribution" "", then publish the application
  10. Click the link "Manage list of testers" and copy the link which should be shared with testers, then forward to the link to become a tester
  11. Click the link "View in Google Play store", then click the button "Add to Wishlist"
  12. Launch the webapp "google play store" in test device
  13. Login in the webapp "google play store" and choose Wishlist
  14. Install the application which you publish
  15. Launch the webapp which you publish
