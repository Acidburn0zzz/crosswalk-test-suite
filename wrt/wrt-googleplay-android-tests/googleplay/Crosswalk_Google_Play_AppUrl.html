Test steps:

  1. Package an web app with command: python make_apk.py --name=Test --package=org.xwalk.test --app-url=http://www.baidu.com --app-versionCodeBase=1 --app-version=1.0.0 --arch=x86
  2. Login the Google Play Developer Console in Browser and click the button "Add new application"
  3. Input a Title and click the button "Upload APK"
  4. Choose ALPHA TESTING or BETA TESTING, then upload the "x86" arch apk into this application
  5. Add the google groups of users for each test
  6. Publish this application
  7. Download the application which you have published form Google Play and install it
  8. Launch the webapp

Expected Output:

  1. Installation package for x86 platform generated successfully
  2. Dialog "ADD NEW APPLICATION" popuped successfully
  3. Application generated successfully with the name which you input
  4. Apk could be uploaded into the application successfully
  5. Add google groups successfully
  6. Application could be published successfully
  7. Webapp could be installed successfully
  8. Webapp could be launched successfully
