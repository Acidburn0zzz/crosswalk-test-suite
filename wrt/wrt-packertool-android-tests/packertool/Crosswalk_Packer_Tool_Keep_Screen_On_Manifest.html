Test steps:

  1. Package an web app with keep-screen-on option, command like: python make_apk.py --manifest=/path/to/manifest.json --keep-screen-on
  2. Install the apk packaged
  3. Set the device screen off as 30 seconds(Setting->Display->Sleep: 30 seconds)
  4. Launch the webapp
  5. Check the screen

  1. Apk could be created correctly
  2. Web app could be installed successfully
  3. The device screen off after 30 seconds
