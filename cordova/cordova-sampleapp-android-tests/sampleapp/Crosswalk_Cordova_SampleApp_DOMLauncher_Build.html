Pre-condition

  • The eclipse ADT has been installed on host
  • Connect the device to host with USB cable

Test Steps

  1. Run the following commands:
    • $ git clone https://github.com/AricwithanA/DOMLauncher2.git
    • $ unzip crosswalk-cordova-<version>-<architecture>.zip
  2. Launch eclipse ADT
  3. Run step: 'File'--> 'Import' --> 'Android' --> 'Existing Android Code Into Workspace' --> 'Browse Root Directory' --> 'Finish'
    1. Import projects from crosswalk-cordova-<version>-<architecture>/framework, include 'framework' and 'xwalk_core_library'
    2. Import project from DOMLauncher2/DOMLauncher/platforms/android
  4. Right click the project 'DOMLauncher' in eclipse package explorer, click 'preperties' --> 'Android'--> 'Add', add "framework" as the referenced project
  5. Right click the project 'DOMLauncher' in eclipse package explorer, click 'Run As' --> 'Android Application'
  6. Click home button in device and select DOMLauncher

Expected Results

  1. Able to clone code and unzip zip package
  2. Launch eclipse ADT successfully
  3. Run the step successfully, no error
    1. The two project can be imported to eclipse
    2. The project can be imported to eclipse
  4. Run the step successfully, no error
  5. The app can be installed on device
  6. The app can be launched
