Test Purpose:

Verifies the functionality of hybrid application. Tests the functionality of two hybrid applications installation, uninstallation, launching, update and communicating with the native application by clicking the "START/ STOP" buttons, clearing the messages by clicking the "CLEAR" button. And quit the hybrid application by clicking the "RETURN" button.

  1. Click the "Install" button to install the Hybrid app.
  2. Click the "Launch" button to launch the Hybrid app.
  3. Click the "Update" button to update the Hybrid app.
  4. Click the "Launch" button to launch the Hybrid app.
  5. Click the "Uninstall" button to uninstall the Hybrid app.
  6. Click the "Try to Launch" button to launch the Hybrid app, it will get an error msg.

Expected Result:

Test passes if all the test points work well.
