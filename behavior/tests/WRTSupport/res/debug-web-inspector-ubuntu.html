• Pre-condition
    • Make sure SDK is installed to PC(ubuntu) and it is launched;
    • Turn on device USB debugging mode.
  • Test step
    1. Open SDK, File->New->Tizen Web Project. Click "Basic", then input Project name, e.g. test-debugging, then click "finish".
    2. Connect test device to PC. Open Window->Show View -> Connection Explorer, the device information appears in the "Connection Explorer" of SDK.
    3. Right-click on the folder "test-debugging" -> Debug as -> 1 Tizen Web Application.
  • Expected Result
  • Test passes if "web inspector" interface for widget debugging is displayed on the screen.

Test Purpose:

Verifies 'web inspector' supported on PC(ubuntu).
