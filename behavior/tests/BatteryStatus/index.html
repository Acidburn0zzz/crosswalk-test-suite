Test purpose

Verifies the functionality of battery status, event work well while charging, discharging, plug and unplug charger, battery full.

Test battery default values

    Pre-conditions:
    1. The implementation is unable to report the battery's charging state, charging time, level or remaining time respectively.
    2. The device is unplugged from the charger before this test case is run
    Test steps:
    1. Launch 'Test battery default values'

Test battery discharging

    Pre-conditions:
    1. The device is unplugged from the charger before this test is run.
    2. The battery must neither be empty or full, nor reach empty or full capacity during the test.
    Test steps:
    1. Launch 'Test battery discharging'

Test battery plugged

    Pre-conditions:
    1. The device is unplugged from the charger before this test is run.
    2. The battery must not be full or reach full capacity during the time the test is run.
    Test steps:
    1. Plug in the charger and wait for all the tests to complete.
    2. Launch 'Test battery plugged'

Test battery unplugged

    Pre-conditions:
    1. The device is plugged in to the charger before this test is run.
    2. The battery must not be full or reach full capacity during the time the test is run.
    Test steps:
    1. Unplug the charger and wait for all the tests to complete.
    2. Launch 'Test battery unplugged'

Test battery charging

    Pre-conditions:
    1. The device is plugged in to the charger before this test is run.
    2. The battery must neither be empty or full, nor reach empty or full capacity during the test.
    Test steps:
    1. Launch 'Test battery charging'

Test battery full

    Pre-conditions:
    1. The device is plugged in to the charger before this test is run.
    2. The battery is full.
    Test steps:
    1. Launch 'Test battery full'

Expected Result

The tests all pass.
