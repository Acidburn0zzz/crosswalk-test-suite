    Check if a notification is successfully posted


    # Simple Notification

    1. Install NotificationTest

    (This application is used to verify AppControl in Notification.)

    2. Click SIMPLE Notification

    3. Check notification tray if a simple notification is posted

    4. Clicks the simple notification and check if a NotificationTest application is launched

    5. In the mean time, check if the notification is disappeared


    # Ongoing Notification

    1. Click ONGOING Notification

    2. Check notification tray if a ongoing notification is posted

    3. Clicks the ongoing notification and check if it is NOT disappeared

    4. Remove the ongoing notification by clicking Delete in the Notification List


    # Progress Notification

    1. Click PROGRESS Notification

    2. Check notification tray if a progress notification is posted

    (The progress notification shows a bar that shows the current progress)

    3. Clicks the progress notification and check if it is NOT disappeared

    4. Remove the progress notification by clicking Delete in the Notification List


    # Thumbnail Notification

    1. Click THUMBNAIL Notification

    2. Check notification tray if a thumbnail notification is posted

    3. Clicks the thumbnail notification and check if it is disappeared
