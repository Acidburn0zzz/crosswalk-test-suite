Test Purpose:

Verifies the functionality of DynamicBox application: mouse event and visibility event for DynamicBox.

Test Step:

  1. Click the "Install" button, test step passes if all app widgets installed successfully.
  2. Click the "Launch" button to launch the DynamicBox Viewer. After each step below, press "Back" key (soft or hard key) to go forward next step:
    1. Click the "Mouse event false"; press the filled blue square; test step passes if the blue square doesn't change;
    2. Click the "Sample"; test step passes if there is a green 'Pass' text;
    3. Click the "Dynamic Box"; press the menu button to select size 354x354; then drag down the box; test step passes if there apears a button "Send message to Box", and click the button, "Hello, Dynamic Box" appends into the box;
    4. Click the "AppWidget Content mouse event"; press the filled blue square; test step passes if the blue square changes to green;
    5. Click the "AppWidget Load event"; test step passes if there is a green 'Pass' text;
    6. Click the "Visibility widget"; test step passes if there is a green 'Pass' text, and after drag down the 'Pass' there appears 'PD index.html';
    7. Click the "Size Decoration"; press the menu button to select box size; test step passes if the box in '354x354' size has different frame decoration from those in '175x175' and '345x175' sizes;
    8. Click the "Size Preview"; press the menu button; test step passes if there are icons in the size list.
  3. Click the "Uninstall" button, test step passes if all app widgets uninstalled successfully.

Expected Result:

Test passes if all the test steps pass.
